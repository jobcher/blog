## 通过docker快速部署
```sh
# 安装docker
curl -sSL https://get.daocloud.io/docker | sh
# 开机docker 自启
systemctl enable docker
systemctl start docker
# docker检查
docker version
# 运行博客
sh updatedocker.sh
```
## 代码结构
```sh
博客文章目录：content\posts
LoveIt配置目录：config.toml
```
