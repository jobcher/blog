<p><b>日本银行计划在周二结束利率曲线控制和购买风险资产，日本货币政策震荡中也结束负利率</b><br><br>东京——《日经新闻》获悉，日本银行准备在周二决定结束其利率曲线控制和购买风险资产，并停止实行负利率政策，这将是日本货币政策近20年来最大的变化。<br><br>中央银行将于周二召开为期两天的政策委员会会议，准备逐步收紧其庞大的货币宽松政策。<br><br>日本银行行长植田和夫和其他政策委员将根据周一的经济和价格数据讨论结果做出决定。随着日本央行的可持续2%的通胀目标即将实现，该银行正首次在17年后推动利率上调。<br><br>日本经济多年来一直在与通缩作斗争，而如今消费者价格上涨，这个决定已经被日本首相岸田文雄的亲信所证实。这一决定“掌握在他们手中”。<br><br>一个主要的提议是将目前的短期政策利率目标从-0.1%提高到0至0.1%之间。自2016年以来，日本央行对一些商业银行在日本银行存放的资金实行负利率，以刺激贷款发放。<br><br>日本央行还将取消利率曲线控制政策，这是2016年9月引入的政策，根据该政策，它购买了大量的日本国债，以使短期收益率保持在-0.1%的目标附近，长期收益率维持在零附近。<br><br>这些购买使得日本银行持有的日本国债总额超过了所有未偿还国债的一半，不包括短期国库贴现票据。<br><br>尽管央行将继续购买一些日本国债以防止债券收益率飙升，但它将放弃控制市场趋势下利率的框架。该银行将取消对10年期收益率的1%参考上限，使其根据市场情况自由变动。<br><br>购买日本股票交易所交易基金（ETF）和房地产投资信托也将结束。<br><br>这项政策始于2010年，当时的央行行长是白川方明，他的继任者黑田东彦加大了这一政策的力度。据央行称，截至9月底，日本银行持有的ETF总价值约为37万亿日元（合2480亿美元），股市反弹使其未实现收益约为30万亿日元。<br><br>日本央行自2022年6月以来就没有购买日本房地产投资信托的股票了。<br><br>最近日本央行官员的评论表明，政策制定者认为，服务业价格上涨已经接近他们希望的水平。<br><br>终止负利率政策的最重要因素之一是工会和雇主之间正在进行的工资谈判，据植田称。上周报告的初步结果显示，平均工资增长率为5.28%，是33年来的最高水平。<br><br>来源：<a href="https://asia.nikkei.com/Economy/Bank-of-Japan/BOJ-to-end-yield-curve-control-and-ETF-purchases-Tuesday" target="_blank" rel="noopener" onclick="return confirm('Open this link?\n\n'+this.href);">日经亚洲</a></p>
