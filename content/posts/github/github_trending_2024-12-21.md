<p>截至2023年底，全国退休人数已经超过3.1亿；其中城镇占45%，城乡占55%。一年养老金支出7.3万亿中；城镇占93%，城乡只占7%。 <br><br>如果把城乡退休待遇提高至城镇的水平，则需增加财政补贴7.85万亿；而2023 年财政两本账赤字已经是8.85万亿；这就意味着财政两本账赤字会达到16万亿，而去年一年的税收收入只有15.9万亿。而如果我们将医保自费比例提升至日本的水平，大概需要每年补贴近1万亿元。<br><br>在我们谈论财政紧缩或者财政宽松时， 要注意这是一个结构性的问题，即结构性对弱势群体的财政紧缩，导致了消费的长期中枢不断下行。<br><br>如果未来宏观向民生(弱势)倾斜财政资源，需要一年8-10万亿(再包含生育、失业等保障)的投入。<br><br>它的好处是会降低居民储蓄率，盘活存量货币流动性。它的成本是，需要挤出其他非民生的资源侵占，大幅降低体制内的含金量。过去多年，在结构性财政紧缩的情况下，债务率攀升后已经和美国相当，其实民生保障应该提前多年做。<br><br> <a href="https://mp.weixin.qq.com/s/dLZpdH46hz1lIjQgx8pTvw" target="_blank" rel="noopener" onclick="return confirm('Open this link?\n\n'+this.href);">mp.weixin.qq.com</a></p>
