<p>租税制度主要由三大系统：收益税、消费税和转移税构成，且各有弊端，书中集中研究田赋、 关税与盐税的复杂问题，揭露田赋附加税的泛滥、关税倾斜对本土经济的侵害，以及盐税专商制度对普通民众生活的直接压榨，竭泽而渔式的税收不仅几乎摧毁社会经济的基本运转，更成为孕育帝国财政危机的温床。租税制度的主要问题在三点：一是税收结构不合理，物的税过多，而人的税不足；二是系统不明确，重复课税的现象普遍存在；三是附加税过重，漫无限制的名目加剧社会不公和经济凋敝，逆向分配使穷人负担更重、富人避税更易。<br><br>租税经济是财政的核心，租税问题也是财政问题中最重要的问题。税收制度是平衡社会资源的重要工具，更应是通过行政力量关注每一个普通人的生存尊严，减少交易费用、降低税负来鼓励长期投资显然更有利于政治稳定。当制度不明确、不稳定时，由于不能形成稳定的预期，结果必然是机会主义和短期行为盛行；当制度不合理、不统一时，则是在鼓励搭便车，或者上有政策下有对策；当制度不公允、不及时时，大量偏离设计初衷的负激励行为和结果涌现。统治主体和纳税主体的良性互动推动国家制度的成长。<br><br><a href="https://weibo.com/5239757203/5116352170034765" target="_blank" rel="noopener" onclick="return confirm('Open this link?\n\n'+this.href);">source</a></p>
